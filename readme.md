[expo + next setup guide](https://docs.expo.io/versions/latest/guides/using-nextjs/)

## Script

`expo init my-project` (typescript option)

`yarn add -D @expo/next-adapter`

`yarn next-expo`

In `package.json` replace the `web` npm script with `next dev`
